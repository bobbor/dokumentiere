private prettify(desc)

transforms `, * and _

parses the description of a function or module
it transforms those to code, strong and em
`foo` == <code>foo</code>
*foo* == <strong>foo</strong>
_foo_ == <em>foo</strong>

Parameter

desc

  • string

the string to parse

Returns

 

  • string

the prettified string

Usage
 var myCodeString = prettify(myString);
